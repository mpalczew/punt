# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/),
and this project adheres to [Semantic Versioning](https://semver.org/).

## [0.1.0] - 2026-03-03

### Added
- Initial release
- Browser picker with floating panel UI
- Browser discovery via LSCopyAllHandlersForURLScheme
- Chromium profile support (Chrome, Brave, Edge, Vivaldi, Opera, Arc)
- Keyboard navigation: number keys for browsers, type to fuzzy-filter profiles
- Domain-based routing rules
- URL tracking parameter stripping
- Menu bar icon with settings access
- Auto-update from GitHub releases
- Login item registration on first launch
- Homebrew cask distribution
